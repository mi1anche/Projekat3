Koristeći Rx.NET i OpenLibrary API, prikupiti informacije o autorima i njihovim knjigama
koristeći opciju pretrage po autoru. Rx vrši osnovno mapiranje dobijenih podataka i emituje ih kao
poruke aktorima. Aktori čuvaju evidenciju o knjigama po autoru kao interno stanje i ažuriraju
podatke o nazivu, godini izdavanja, jezicima prevoda i prosečnom rejtingu svake knjige. Web
server prima zahtev sa imenom autora i prevodi ga u poruke aktorima. Prikazati dobijene rezultate.
Dokumentacija dostupna na linku: https://openlibrary.org/dev/docs/api/search
